User Stories from Product Backlog that we will address in this sprint:
a) As a user, I have access to a file system of files and directories so that I can work with my data
b) As a user, I can see the prompt for input so that I know when to input.

d) As a user, I can get back an audible form of what I typed so that I can hear my input. (speak)
